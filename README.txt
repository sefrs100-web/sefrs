
# TravelCoach PWA 풀세트
이 폴더를 그대로 GitHub Pages / Netlify / Vercel에 업로드하면,
안드로이드 크롬에서 **홈 화면에 설치** 버튼이 활성화됩니다.

## 업로드 요약
- GitHub: 저장소 만들기 → 파일 업로드 → Settings > Pages > main/root → 주소 발급
- Netlify: Add new site > Deploy manually → 폴더 드래그 → 주소 발급
- Vercel: New Project > Deploy → 폴더 업로드 → 주소 발급

※ 반드시 HTTPS 주소에서 접속해야 설치 버튼이 보입니다.
